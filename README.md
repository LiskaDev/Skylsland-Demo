# SkyIsland Demo (空中岛屿建造原型)

一个基于 Unity 开发的 3D 沙盒建造类游戏 Demo。玩家可以在悬浮的岛屿上进行方块建造，并指挥村民自动化建设。

## 核心玩法 (Core Gameplay)
- **第一人称探索**：玩家可以在岛屿间移动、跳跃。
- **沙盒建造**：支持多种材质方块（石材、木材）的网格对齐放置与拆除。
- **村民自动化**：通过蓝图系统指派村民前往指定地点建造复杂结构（小屋、塔、围墙）。
- **特色交通**：通过纸飞机系统进行岛际穿梭（开发中）。

## 快速开始 (Quick Start)
1. **运行环境**：Unity 2021.3+ (建议版本)。
2. **场景位置**：`Assets/Scenes/SampleScene.unity`。
3. **操作说明**：
   - `WASD`：移动 | `Space`：跳跃。
   - `数字键 1-3` 或 `鼠标滚轮`：切换建造方块类型。
   - `鼠标左键`：放置方块 | `Ctrl + 左键`：拆除方块。
   - `E键`：靠近村民打开/关闭蓝图菜单。
   - `鼠标右键`：指派村民建造选中的蓝图。
